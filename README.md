# Sandy Pyramid Maze

A Super Mario Bros–themed maze game where a fighter navigates a sandy pyramid dungeon, collecting bones scattered on the floor before escaping through the golden exit door.

## Setup and Interaction Instructions

1. Open the repository in VS Code.
2. Right-click `index.html` and select **Open with Live Server** (or use the Live Server extension).
3. The game will open in your default browser — use **Google Chrome** for best results.

**Controls:**
- `W` — move up
- `S` — move down
- `A` — move left (sprite flips to face left)
- `D` — move right

**Goal:** Collect all 3 bones on the dungeon floor, then navigate to the golden exit door in the bottom-right corner.

**Opening the Chrome Console (for debugging):**
- Mac: `Cmd + Option + J`
- Windows: `F12` → Console tab
